How to Apply CSS to HTML

There are three ways to apply CSS to HTML:
1. Inline - by using the style attribute inside an HTML element

                
                    <p style="background-color:grey;"> Hello World!</p>
2. Internal - by using a <style> element in the <head> section
                
                    <head>
                        <style>
                            body {
                                background-color: rgb(30, 30, 30);
                            }
                        </style>
                    </head>
3. External - by using a <link> element in the <head> section to link to an external CSS file
                
                    <head>
                        <link rel="stylesheet" href="path/to/your/styles.css">
                    </head>

How to Use JavaScript in a Webpage

To link an external JavaScript file you need to put a <script></script> element in the <head> section 

                
                    <script src="path/to/your/script.js"></script>
To write a to write JavaScript code within an html file the JavaScript code must be between <script> element rather than providing a src attribute 
                
                    <script>
                        window.addEventListener("load", () => {
                            console.log("This function is executed once the page is fully loaded");
                        });
                    </script>
